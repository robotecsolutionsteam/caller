## Importação

<Row>
  <Col>
    Importar o Caller.
  </Col>
  <Col>

    ```bash {{ title: "kotlin" }}
    import com.robotec.caller.movements.Movements 
    ```
  </Col>
  <Col>
    Colocar o Movements em variável global.
  </Col>
  <Col>

    ```bash {{ title: "kotlin" }}
    private val movements = Movements()
    ```
  </Col>
</Row>
