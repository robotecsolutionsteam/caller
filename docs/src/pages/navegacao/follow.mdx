# Follow Me - Seguir Pessoas
---

### Importação
Como utilizar as funções de Follow Me.

### Follow me
Ativar o Follow Me, normal sem nenhuma restrição. Poser ser utilizado offline.

### Follow Restrito
Ativar o Follow Me, com restrição de movimento, onde o Temi não consegue do lugar, somente seguir com a cabeça.

### Listener Follow Me
Utilize o listener **currentFollowStatus** para receber o status do follow me.

## Importação

<Row>
  <Col>
    Importar o FollowMe. No inicio do arquivo.
  </Col>
  <Col>
    ```bash {{ title: "kotlin" }}
    import com.robotec.caller.follow.FollowMe
    ```
  </Col>
  <Col>
    Colocar o Follow Me em variável global.
  </Col>
  <Col>


  </Col>

## Follow Me

<Row>
  <Col>
    Iniciar follow me.
    #### Parâmetros
    * useTemi: **Boolean**.

    #### Retornos
    * **None**
  </Col>
  <Col>
