Deletar um local salvo.

## Importação

<Row>
  <Col>
    Importar o Caller.
  </Col>
  <Col>
    ```bash {{ title: "kotlin" }}
    import com.robotec.caller.navigation.Navigation
    ```
  </Col>
  <Col>
    Colocar o Navigation em variável global.
  </Col>
  <Col>
    ```bash {{ title: "kotlin" }}
    private val navigation = Navigation()
    ```
  </Col>
</Row>
