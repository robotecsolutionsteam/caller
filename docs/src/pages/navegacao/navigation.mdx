# Sistema de Navegação
Goto, save local, etc.
---

### Importação

* Salvar Local
* Deletar Local
* Obter Locais Salvos
* GoTo Simples
* GoTo Completo
* GoTo Posição
* Reposicionar

## Importação

<Row>
  <Col>
    Importar o Caller.
  </Col>
  <Col>

    ```bash {{ title: "kotlin" }}
    import com.robotec.caller.navigation.Navigation
    ```
  </Col>
  <Col>
    Colocar o Navigation em variável global.
  </Col>
  <Col>

    ```bash {{ title: "kotlin" }}
    private val navigation = Navigation()
    ```
  </Col>
</Row>

## Salvar Local

<Row>
  <Col>
      Chamar o método **saveLocal**.
      ### Parâmetros
      * local ->  **String**.

      ### Retornos
      * None
  </Col>
  <Col>

    ```bash {{ title: "kotlin" }}
    navigation.saveLocal(local) {
      // Código de callback
    }
    ```
  </Col>
</Row>

## Obter Locais Salvos

<Row>
  <Col>
      Chamar o método **getlocations**.
      ### Parâmetros
      * None

      ### Retornos
      * locals -> **List String**
  </Col>
  <Col>

    ```bash {{ title: "kotlin" }}
    val locals = navigation.getLocations()
    ```
  </Col>
</Row>

      ### Retornos

  </Col>
  <Col>

## GoTo Completo

<Row>
  <Col>
      Chamar o método **gotocomplex**.
      ### Parâmetros
      * local -> String
      * Ré -> Boolean
      * Sem Desvio -> Boolean

      ### Retornos
      * None
  </Col>
  <Col>

    ```bash {{ title: "kotlin" }}
    navigation.goToComplex(local, backwards, noBypass) {
      // Código de callback
    }
    ```
  </Col>
</Row>

<Row>
